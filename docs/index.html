OpenDXL Streaming Java Client Library¶

See the navigation bar on the right for an overview of the OpenDXL Streaming Java Client Library and examples.

Overview¶

The OpenDXL Streaming Java client library is used to consume records from a Data Exchange Layer (DXL) Streaming Service.

The DXL Streaming Service exposes a REST-based API that communicates with a back-end streaming platform. The streaming service performs authentication and authorization and exposes methods to retrieve records.

One concrete example of a DXL Streaming Service is the McAfee MVISION EDR “Events feed”.

Installation¶

To start using the OpenDXL Streaming Java client library:

  • Download the Latest Release
  • Extract the release .zip file
  • View the README.html file located at the root of the extracted files.
    • The README links to the documentation which includes installation instructions and usage examples.
    • The Javadoc API documentation is also available on-line here.
